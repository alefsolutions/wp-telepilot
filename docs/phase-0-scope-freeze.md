# TelePress Phase 0 Scope Freeze

## Included in the hardening window

- Security fixes
- capability-check corrections
- transport reliability improvements
- onboarding clarity improvements
- documentation
- diagnostics
- admin UX polish
- essential Telegram workflow improvements for the core modules

## Not included in the hardening window

- WooCommerce
- AI features
- theme design tools
- navigation menu builders
- widget management
- multi-site cloud orchestration
- major new integration surfaces

## Release intent

Phase 0 exists to make TelePress dependable, explainable, and safe enough to support the first public Core release.
